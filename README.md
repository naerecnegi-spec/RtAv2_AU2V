# RtAv2 (Pure Lua Edition) for AviUtl2

REAPER のプロジェクトファイル（`.rpp`）を読み込み、タイムライン上のオブジェクト（画像やカスタムオブジェクトなど）をBPMや音符のタイミングに合わせて自動でアニメーション（反転、移動、回転、拡大、透明度変更）させる AviUtl2（64-bit版）用のプラグイン/スクリプトです。

オリジナル（無印版）の C++ DLL による実装を **Pure Lua + FFI** で完全に書き直し、64-bit版の AviUtl2 (ExEdit2) 環境に対応させました。

---

## ✨ 主な特徴

- **64-bit 完全対応**: 従来の `RtAv2.dll` は 32-bit 版であったため AviUtl2 では動作しませんでしたが、Pure Lua と FFI 呼び出しで再構築したことにより、DLL 不要で AviUtl2 で動作します。
- **日本語（マルチバイト）パス対応**: Windows API（`kernel32` / `ucrtbase`）のワイド文字（UTF-16）ファイルオープンを LuaJIT FFI を用いて呼び出すため、日本語が含まれるパスの `.rpp` ファイルやプロジェクトフォルダでもクラッシュせずに正常にロードできます。
- **オブジェクトエイリアスのフォルダ化対応**: 各オブジェクト（反転、座標、回転など）を `C:\ProgramData\aviutl2\Alias\RtAv2` に綺麗にグループ化して配置するため、AviUtl2 のタイムライン上の追加メニュー「RtAv2」フォルダから簡単に追加できます。
- **自動インストーラー同梱**: パイソンによる配置スクリプト `deploy_final.py` を同梱しており、実行するだけでファイルコピー、エイリアス変換、`aviutl2.ini` への登録までを自動で行います。

---

## 📦 ファイル構成

- **`RtAv2.lua`**: `.rpp` 解析コア（Pure Lua & Windows FFI IO）
- **`RtAv2Function.lua`**: 座標計算・共有バッファマネージャー
- **`@RtAv2.anm`**: 初期化・描画・rpp読み込みフィルタ
- **`@RtAv2(エフェクト).anm`**: 反転・座標・回転・拡大率・透明度エフェクト
- **`@RtAv2(描画系).anm`**: 複数描画・両端クリッピングなどの描画拡張
- **`@RtAv2(読込系).anm`**: 動画読み込み・エフェクトリセット

## 🚀 導入手順

### 方法 1: 自動デプロイ（推奨）

Pythonがインストールされている環境であれば、最も簡単で確実です。

1. 本リポジトリをダウンロード（またはクローン）します。
2. コマンドプロンプトまたは PowerShell を **管理者として実行** します。
3. リポジトリのディレクトリで以下のコマンドを実行します：
   ```bash
   python deploy_final.py
   ```
4. `Deployment successfully completed!` と表示されれば完了です。AviUtl2 を起動すると、エイリアスメニューに `RtAv2` フォルダが追加されています。

### 方法 2: 手動で配置

1. `RtAv2.lua`, `RtAv2Function.lua` を AviUtl2 のスクリプトフォルダ（例: `C:\ProgramData\aviutl2\Script`）にコピーします。
2. アニメーション効果ファイル（`@RtAv2*.anm`）を同様にスクリプトフォルダにコピーします。
   ※ 必要に応じて、AviUtl2のスクリプトインポート用拡張子 `.anm2` にリネームして配置してください。
3. エイリアスファイル（`.object`）を作成し、`C:\ProgramData\aviutl2\Alias\RtAv2` フォルダ内に配置します。

---

## 🛠️ 使い方

1. REAPERで音楽プロジェクトを編集し、`.rpp` ファイルとして保存します。
2. AviUtl2で画像や図形などのオブジェクトを作成します。
3. オブジェクトに以下のフィルタを順に追加します（タイムライン右クリック ➔ フィルタ効果の追加）：
   - `初期化@RtAv2`
   - 各種エフェクト（`座標@RtAv2`、`反転@RtAv2` など）
   - `描画@RtAv2`
4. タイムライン上の `初期化@RtAv2` に、REAPERの `.rpp` ファイルのパスを読み込ませることで、再生に合わせて自動的に動き出します。
